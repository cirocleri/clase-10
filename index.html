¿Qué toca hoy? (completar)

Hoy toca Git y Github, 2 cosas muy interesantes.

Reveer la página, a partir del descanso, o que empieza a ver cosas de git en si, colgué un poco acá

Git

Git sirve como "librería" o repositorio para guardar las versiones que fuí haciendo de un código, esto es de manera meramente LOCAL, para ONLINE o compartir vamos a verlo más adelante (github). Nos permite ir variando e intercalando entre las distintas versiones de nuestro código, ya sea para verlos, editarlos o lo que queramos.

GITHUB

Github es "la red social de los ñoños", va de la mano con git, es una red donde usamos, compartimos, corregimos y vemos el cógido de los demás, como los demás pueden ver, corregir y todo lo que sea respecto a nuestros códigos. Sirve mucho para dar y recibir Feedback de códigos.
